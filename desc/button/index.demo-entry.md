```demo
basic

```

## Props

### Button Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| icon | `string` | `null` | 按钮图标 |
| iconPosition| `left` \| `right` | `left`| 按钮图标的位置 | 
| loading | `boolean` | `false` | 按钮是否显示加载状态|
| type | `default` \| `primary` \| `success` \| `info` \| `warning` \| `error` | `default`  | 按钮的类型 |
