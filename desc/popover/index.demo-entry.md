# 弹出信息 Popover

Popover 在内容周围弹出一些隐藏的信息。Popover 里面没什么内置样式，在里面填什么主要靠你了。

## 演示

```demo
basic
```

## Props

### popover Props

| 名称 | 类型 | 可选值 | 默认值 | 说明 |
| --- | --- | --- |  --- | --- |
| position | `string`| `top` \| `bottom` \| `left` \| `right`| `top`  | popover 的弹出位置 |
| trigger | `string` |`click` \| `hover` | `click` | popover 的触发方式 |
