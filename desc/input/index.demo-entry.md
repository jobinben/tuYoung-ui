## Props

### Button Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| type | `text` \| `password` \| `number` \| `textarea`| `text`  | 输入框类型 |
| value | `string` | `undefiend` | 文本输入的值 |
| placeholder| `string ` | `undefiend`| 文本输入的占位符 | 
| disabled | `boolean` | `false` | 是否禁用|
| error | `string` | `undefiend` | 输入错误提示 |
| success | `string` | `undefiend` | 输入正确提示 |
