
# 一个基于Vue的组件库 -- tuYoung 

​	作者： 大冰


## 前期准备

1. vue-cli 安装

2. 测试库使用chai（断言库）mocha（自动化测试）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
